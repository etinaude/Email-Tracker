# Tracker

A basic email, website and github tracker

This is currently running on my VPS but can be run on your own.

## Installing

```bash
git clone https://github.com/etinaude/tracker.git
cd tracker
npm install
```

## Running

```bash
npm start
```

## Technologies

This uses:

- Node & Express JS backend
- SQL database
- Travis CI
- Vanilla JS front end
